# Rating-Mini-Program
这款评分微信小程序，广泛适用于各类评价场景。商家能用它收集顾客对商品或服务的反馈，简单几步创建评分项目，涵盖产品质量、服务态度等维度，设定星级或量表评价形式。顾客扫码进入小程序，轻松打分并可附上文字、图片评价，让反馈更直观。教育领域，教师用它开展学生互评、自评，促进教学相长。活动组织者能借此收集参与者对活动的评价，优化后续活动。小程序自动统计分析数据，生成可视化报表，为决策提供有力依据。
